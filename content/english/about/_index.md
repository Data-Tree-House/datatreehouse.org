We're two developers, and two friends from work.

datatreehouse.org is our shared workshop. A place where we build things, break things, and figure out how it all works along the way. We're drawn to Streamlit apps, data pipelines, scraping projects, and whatever else catches our eye on any given week. If it's interesting, it's fair game.

### What we're about

- We self-host everything we possibly can
- We build things primarily to learn, not to ship
- We're comfortable not knowing the answer yet
- We like data — collecting it, moving it, visualising it
- We document as we go, so future-us (and maybe you) can follow along

We're not here because we have all the answers. We're here because we love the questions. Every project is an excuse to pick up something new — a tool, a concept, a rabbit hole worth falling down.

If something we've built or written has been useful to you, that's a bonus. Pull up a branch and stay a while.
